# Origin Web Traditional Chinese - Phase 3

This package extends the working Phase 2 project.

## New Chinese routes

- `/zh/contact`
- `/zh/book-consultation`
- `/zh/thank-you`
- `/zh/privacy-policy`
- `/zh/terms-of-use`
- `/zh/professional-disclaimer`

## Improvements

- Chinese cookie banner text and buttons
- Chinese legal consent wording on the enquiry form
- Chinese form labels, validation messages and confirmation page
- Chinese navigation and footer links updated
- Removed the development-style Back bar from all Chinese pages
- Existing English routes and services remain unchanged

## Run locally

```powershell
npm install
npm run dev
```

Open `http://localhost:5173/zh`.

## Build test

```powershell
npm run build
```
